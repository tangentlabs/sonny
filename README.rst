Wolseley Importing
====

All the importing that needs to be done for Wolseley


Usage
====

It's enough to do:

``python -m path.to.importer.package``

To pass arguments:

``python -m path.to.importer.package run arg1 arg2``

To test a job:

``python -m path.to.importer.package test arg1 arg2``


Tests
====

It's enough to do:

``nosetests``
